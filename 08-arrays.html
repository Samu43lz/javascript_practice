Arrays (Arreglos)

Un arreglo (o array, en inglés) es como una caja organizadora con varios compartimentos. En cada uno de esos compartimentos, puedes guardar un valor. Piensa en un arreglo como una fila de casilleros, donde cada casillero tiene un número (su posición/Indice) y puede contener un valor diferente, como un número, una palabra, o cualquier otro tipo de dato.

Métodos

Array.of

El método Array.of crea un nuevo array con los valores que le pasas. A diferencia de otras formas de crear arrays, como usando corchetes ([]) o el constructor Array, Array.of es más predecible y directo cuando estás creando un array con un solo valor numérico.

En resumen Array.of es una forma confiable y directa de declarar un array en JavaScript.

Array.fill

El método Array.prototype.fill en JavaScript se utiliza para llenar todos los elementos de un array con un valor estático, desde un índice inicial hasta un índice final opcional. Es una forma práctica de inicializar o sobrescribir los elementos de un array con un valor específico.

En la imagen estamos creando 100 espacios y en cada uno le estamos colocando el valor booleano de "false".
