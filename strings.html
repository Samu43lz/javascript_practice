Cadenas de texto (Strings)

Recuerda revisar la consola para ver como actuan!

¿Como insertarlas?

Podemos añadirlas con comillas simples o dobles.

Objetos String

Una forma de crear un objeto de tipo "string" utilizando el constructor "string".

Propiedad length

La propiedad "length" muestra la cantidad de caracteres que tiene nuestro string.

Metodos

Metodo: toUpperCase

Este metodo convierte todos los caracteres de nuestro string en mayusculas.

Metodo: toLowerCase

Este metodo convierte todos los caracteres de nuestro string en minusculas.

Metodo: includes

Este metodo sirve para buscar una palabra en nuestro string. (nos bota true o false)

Metodo: trim

Este metodo quita los espacios en blanco o caracteres que haya en nuesto string.

Metodo: split

Este metodo separa cada palabra de nuestro string en comillas.
