Objetos

Un objeto es una colección de pares de llave y valor muy parecido a lo que hacemos en CSS, dentro podemos colocar muchos tipos de datos(todos). En Javascript es famosa la frase que dice "En Javascript todo es un objeto"

Estructura

Un objeto puede contener cualquier tipo de dato, ademas,dentro de un objeto a las variables se les van a llamar atributos/propiedades y a las funciones se les llama métodos

Funciones dentro de Objetos

Si queremos usar una función y queremos implementar un atributo del objeto en el que estamos debemos usar la palabra clave (keyword) .this

Formas de mostrar en consola

El que se recomienda y es más facil de utilizar es la de colocar el objeto + "." (atributo) y para las funciones en este caso estando dentro de un objeto se les llama "Métodos" se los invoca colocando el nombre del objeto + (el nombre del método) + "()"
